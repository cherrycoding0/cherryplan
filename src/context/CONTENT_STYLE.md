## X(트위터) 글쓰기 규칙

### 말투
- 친근한 존댓말 사용 (~요 체): "~했어요", "~하더라고요", "~해보세요"
- 자연스러운 구어체: 딱딱하지 않게, 친구한테 말하는 느낌
- 공감형 도입부로 시작: "~하는 분들 많더라고요", "저도 처음엔~", "이거 몰랐으면 큰일 날 뻔했어요"
- 강조할 때만 **볼드** 사용 (남발 금지)
- 짧고 끊어지는 문장 구사 (호흡 짧게)

- 번호 + 이모지로 핵심 내용 나열
- 마무리: 행동 유도 or 공감 멘트 + 이모지

## 블로그 글쓰기 규칙

### 말투
- X보다 조금 더 설명체지만, 여전히 친근한 존댓말 유지
- 독자를 "여러분"으로 호칭 가능
- 어렵지 않게: 전문 용어는 쓰되 바로 설명 추가
- 실제 경험담 or "저는~" 형식의 1인칭 서술 포함

### 구조
1. **도입부**: 공감 or 문제 제기 (2~3문장)
2. **본문**: H2/H3 소제목으로 섹션 구분, 각 섹션 3~5문장
3. **실전 예시**: 코드블록 or 스크린샷 설명 포함 권장
4. **마무리**: 핵심 요약 + 다음 액션 or 시리즈 예고

## 사용 방법

### 1. CLAUDE.md에 삽입 (프로젝트 루트)
위 SYSTEM PROMPT 블록을 `CLAUDE.md` 파일에 붙여넣으면 Claude Code가 프로젝트 컨텍스트로 자동 인식합니다.

### 2. 매 요청 시 인라인 삽입
```bash
claude "위 스타일 가이드를 따라서, Claude Code 입문자를 위한 X 트윗 3개 작성해줘. 주제: CLAUDE.md 만들기"
```

### 3. `/content` 슬래시 커맨드로 등록
`.claude/commands/content.md` 파일 생성 후 아래 내용 삽입:
```markdown
체리플랜 스타일 가이드를 따라 콘텐츠를 작성합니다.
$ARGUMENTS
```
이후 Claude Code에서 `/content X 트윗 작성: CLAUDE.md 활용법` 형태로 호출 가능.
