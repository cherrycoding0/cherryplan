# 🍒 CherryPlan 스킬 로드맵

CherryPlan 9개 앱을 만들면서 쌓이는 기술 스택 정리.
앱을 완성할 때마다 체크해가세요.

---

## 스킬 단계별 정리

| 단계 | 스킬                         | 적용 앱        | 왜 중요한가             |
| ---- | ---------------------------- | -------------- | ----------------------- |
| 기초 | **React useState/useEffect** | 전체           | 모든 앱의 기반          |
| 기초 | **Tailwind CSS 반응형**      | 전체           | 빠른 스타일링           |
| 기초 | **localStorage CRUD**        | 독서, 포모도로 | 백엔드 없이 데이터 유지 |
| 중급 | **React Router v6**          | 허브 구조      | SPA 라우팅              |
| 중급 | **useRef + setInterval**     | 포모도로       | 정확한 타이머           |
| 중급 | **Chart.js / Recharts**      | 가계부, 습관   | 데이터 시각화           |
| 중급 | **날짜 계산 (date-fns)**     | 습관 트래커    | 스트릭/히트맵           |
| 고급 | **Fetch + Streaming**        | AI 일기        | Claude API 연동         |
| 고급 | **커스텀 훅 분리**           | 전체 리팩토링  | 코드 재사용성           |
| 고급 | **Netlify Functions**        | AI 일기        | API 키 서버사이드 보호  |

---

## 추천 구현 순서

스킬이 자연스럽게 누적되도록 난이도 순으로 만드는 걸 권장해요.

- [ ] **1. 허브 (완료)** — React Router, 컴포넌트 구조, Tailwind 레이아웃
- [ ] **2. 포모도로** — useRef, setInterval, localStorage (타이머 정확도 핵심)
- [ ] **3. 독서 기록** — localStorage CRUD, 별점 UI, 리스트 필터
- [ ] **4. 오늘의 메뉴** — 랜덤 로직, CSS 애니메이션 (가볍게 연습)
- [ ] **5. 팀 회고 보드** — 복잡한 상태 관리, 다중 컬럼 CRUD
- [ ] **6. 가계부** — Chart.js/Recharts 연동, 날짜별 집계 로직
- [ ] **7. 습관 트래커** — date-fns 날짜 계산, 히트맵 렌더링, 스트릭 알고리즘
- [ ] **8. AI 일기** — Claude API 스트리밍, Netlify Functions로 키 보호
